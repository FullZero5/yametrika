# New Yandex Metrika Plugin for [Flextype](http://flextype.org/) 0.9.0

## Installation
1. Unzip plugin to the folder `/site/plugins/`
2. Go to `/settings.json` and add id yandex metrika counter.
3. Save your changes.


## Settings in file settings.json

```json
{
 "enabled": true, 
 "YandexId": 12345 
}
```
